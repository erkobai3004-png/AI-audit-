\# Social Engineering Audit Tool



\## Description

This project is developed as part of a diploma research focused on corporate security auditing against social engineering threats.



The system simulates phishing and social engineering scenarios in a controlled environment to analyze user behavior and awareness.



\## Features



\- Flask-based backend

\- User behavior tracking

\- Scenario-based audit pages

\- Admin dashboard

\- CSV logging system

\- Excel/PDF export

\- AI-powered analytics

\- Nginx reverse proxy support

\- Cloudflare integration



\## Technologies



\- Python

\- Flask

\- HTML/CSS/JavaScript

\- Chart.js

\- Pandas

\- OpenPyXL

\- ReportLab

\- Nginx

\- Cloudflare
